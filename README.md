# CitaPro

Aplicación móvil multiplataforma desarrollada con Ionic y Angular.

## Autor
Julian Lopez Restrepo

## Descripción
CitaPro es una aplicación para la gestión de citas, clientes y servicios en pequeños negocios como peluquerías, spa y tiendas de mascotas.

## Funcionalidades implementadas
- Inicio de sesión
- Menú principal
- Lista de servicios
- Agendamiento de citas
- Navegación entre pantallas

## Tecnologías
- Ionic
- Angular
- TypeScript
- SCSSy
